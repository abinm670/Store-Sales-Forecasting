Train table¶

This table contains historical data from 2010-02-05 to 2012-11-01 with columns below:

  • Store number: Number of the store
  • Date: From 2010-02-05 till 2012-11-01
  • Weekly sales: amount of sales
  • IsHolidy: its holdy day or not ### features.csv: This table has aditional date contains:
  • Store: Store number
  • Date: From 2010-02-05 till 2013-07-26'
  • Temp
  • Fuel_Price: cost of fuel
  • Markdown from 1 till 5: promotional after Nov 2011
  • GPI: price index
  • Unemployment rate
  • IsHolidy: holdiay

    def wmae_test(test, pred): # WMAE for test
  weights = X_test['Type2'].apply(lambda is_holiday:5 if is_holiday else 1)
  error = np.sum(weights * np.abs(test - pred), axis=0) / np.sum(weights)
  return error
